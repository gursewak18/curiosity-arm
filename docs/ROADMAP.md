# Curiosity Arm Engineering Roadmap

## Stage 0: Environment Setup

- Install Python 3.12
- Install VS Code
- Install Git
- Create virtual environment
- Install project dependencies

## Stage 1: Python Fundamentals

- Variables
- Loops
- Functions
- Classes
- Lists
- Dictionaries
- Debugging

## Stage 2: Robotic Arm Simulation

- Display a 2-link arm
- Add keyboard controls
- Add forward kinematics
- Render joints, links, target, and HUD

## Stage 3: Advanced Robotics Math

- Add inverse kinematics
- Add reach limits
- Add smooth automatic movement
- Add target-based control

## Stage 4: AI Hand Tracking

- Connect webcam
- Track hand landmarks
- Map hand coordinates to arm targets

## Stage 5: Object Detection AI

- Detect objects
- Return object coordinates
- Send detected targets to the arm

## Stage 6: Real Hardware

- Connect Arduino
- Send servo angle commands
- Validate power and wiring
- Test one servo before the whole arm

## Stage 7: Autonomous Robotics

- Add voice commands
- Add pick-and-place routines
- Add path planning
- Add collision checks

## Stage 8: Industrial Robotics Systems

- Learn ROS2
- Move to Linux workflows
- Explore Gazebo, PyBullet, or Isaac Sim
- Study reinforcement learning and machine vision pipelines
